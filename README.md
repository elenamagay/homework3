# **Week 3 Assignment. Password Generator**

## **Description**
### In the third week assignment the students had a task to create an application that can be used to generate a random password based on selected criteria by modifying starter code. This app will run in the browser, and will feature dynamically updated HTML and CSS powered by JavaScript code that the students write.
